## 课程的意义

理解顶层抽象背后的底层原理

* 了解计算机工作的原理
* 对性能的优化有更深刻的认识
* 不同语言之间的区别
* 了解和避免内存泄漏

## 本课程的方法和原理

* 抽象
* 摩尔定律和缩放原理
    * 发展在指数级变化（目前在放缓）
* 局部性原理和存储器层次结构
    * 随着内存的距离核心变远，其存储量越大，价格更便宜，但是显著变慢
* 并行性
    * 计算系统的很多事情同时发生
    * 多个线程
* 性能测量以及根据测量结果改进的可靠性和冗余的反馈
    * 冗余：多数投票来决定答案 降低了错误的风险
