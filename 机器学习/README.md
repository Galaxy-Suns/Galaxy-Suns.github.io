为什么机器学习应用如此广泛？

我们可以对机器进行简单的编程，比如如何找到从a到b的最短路径

但是很多时侯我们不知道如何编写一个程序来做更有趣的事情，比如我们上述提到的应用

但是我们可以让机器自己学会做

目前机器学习还有很多领域中有待应用，因此前景光明
