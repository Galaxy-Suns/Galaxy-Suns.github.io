## 计算

我们遵循三个步骤计算这个面积

1. 分成"矩形"
2. 累加这些矩形的面积
3. 使得矩形变薄，取得其最薄极限

![](img/1c539f4b.png)

有的地方多一部分 有的地方少一部分

但是随着矩形越来越薄，我们漏掉的小面积会趋于0 变得忽略不计
