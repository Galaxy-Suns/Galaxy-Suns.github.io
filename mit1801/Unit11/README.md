## 相关变化率

警察距离路30 我们的车行驶在路上 警察会向我们的车发送雷达 他们读出我们的车距离50 他们还知道我们的车以80/s的速度沿雷达线靠近 我们是否超速？（> 65/s)

![](img/eab79a18.png)

根据勾股定理 下边为40 

**首先了解什么是变化的 什么是不变的**

我们用t表示时间 

汽车到垂足的距离是变化的 我们命名为x

我们所求就是其随着t的变化率

dx/dt 

沿着雷达线的距离也是变化的 我们设置为D

但是边30不变

![](img/cbde0b7b.png)

80就是D的变化率 dD/dt
