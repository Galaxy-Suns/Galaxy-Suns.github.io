## 曲线草图

目标 使用f'和f''绘制f

忘记以前学习微积分的一些直觉：**不要丢掉初等数学的技能和直觉**

微积分仅仅是弥补以前没有接触的一些空白

### 基本原则

* 如果f'(x) > 0 那么f(x) 递增 反之递减
* 如果f''是正数 那么f'(x) 递增 反之递减

![](img/7952eaf2.png)

整个函数递减，因此f'(x)小于0，而越来越缓，说明斜率不断增加 因此f'(x)递增，故f''(x)大于0

* f''(x) > 0 等同于f(x)凸（图中） *注意国内凹凸性和国外相反 这里是国外的叫法*
