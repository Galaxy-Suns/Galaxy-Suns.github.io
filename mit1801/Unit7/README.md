## 自然对数是自然的

### 例子_经济学

如果持有的股票价格下跌 27.9点 （要知道这个数字的原价格 才有意义 比如 6432点）

价格变化ΔP/P = 27.9/6432 = .43%才是有意义的

而取其无穷小量（在投资时人们通常认为天数是一个相对较小的增量）

p'/p = (lnp)' 正是对数微分的公式 因此 自然对数具有实际意义
