# 类和对象
在c++中认为万事万物皆对象
对象具有属性，行为
具有相同性质的**对象**，可以抽象为**类**

c++面向对象的三大特性 **封装、继承、多态**
## 封装
* 属性和行为作为整体表现事物
* 将属性、行为加以权限控制
##### 属性和行为整体表现事物--创建类
```c++
// 设计一个圆类，求圆的周长
// 圆求周长的公式 2*pi*r
const double PI = 3.14; // 圆周率
// class 代表设计一个类 类后面是类的名称
class Circle
{
    // 访问权限
    // 公共权限
public:
    // 属性
    // 半径
    int m_r;

    // 行为
    // 获取圆的周长
    double calculateZC()
    {
        return 2 * PI * m_r;
    }
};

int main()
{
    // 通过圆类 创建具体的圆(对象)
    //实例化:通过一个类创建一个对象的过程
    Circle c1;
    c1.m_r = 10;
    std::cout << "圆的周长为: " << c1.calculateZC() << std::endl;
    return 0;
}
```
##### 将属性和行为加以权限控制
1. public 公共权限
2. pritected 保护权限
3. private 私有权限

```c++
// 访问权限
// 三种
// 公共权限 public 成员 类内可以访问 类外可以访问 子类可以访问
// 保护权限 protected 成员 类内可以访问 类外不可以访问 子类可以访问
// 私有权限 private 成员 类内可以访问 类外不可以访问 子类不可以访问
class Person
{
public:                 // 公共权限
    std::string m_Name; // 姓名
protected:              // 保护权限
    std::string m_car;  // 汽车
private:                // 保护权限
    int m_Password;     // 银行卡密码
public:
    void func1()
    {
        m_Name = "张三";
        m_car = "拖拉机";
        m_Password = 123456;
    }
};

int main()
{
    // 实例化具体对象
    Person p1;
    p1.m_Name = "李四";
    // p1.m_car="奔驰"; 保护权限内容，类外不可访问
    // p1.m_Password=123; 私有权限内容，类外访问不到
}
```

### c++中struct和class区别
仅有默认权限的区别
* struct默认权限是public
* class默认权限是private

而在c++中struct也可有成员函数和属性
```c++
class C1
{
    int m_A; // 默认权限是私有
};

struct C2
{
    int m_A; // 默认权限是公共
};

int main()
{
    C1 c1;
    C2 c2;
    // c1.m_A=100; 不可访问
    c2.m_A = 100;
    return 0;
}
```
### 成员属性设置为私有的优点
1. 可以通过公有方法控制读写权限
2. 在写时可以通过if来判断数据有效性

```c++
// 成员属性设置为私有
// 1、可以自己控制读写的权限
// 2、对于写可以自己检测数据的有效性
class Person
{
public:
    // 写姓名
    void setName(std::string name)
    {
        m_name = name;
    }
    // 获取姓名
    std::string getName()
    {
        return m_name;
    }
    // 获取年龄
    int getAge()
    {
        // m_age = 0; // 初始化为0岁
        return m_age;
    }
    void setAge(int age) // 设置年龄 判断有效性
    {
        if (age < 0 || age > 150)
        {
            m_age = 0;
            std::cout << "输入数据无效" << std::endl;
        }
        else
            m_age = age;
        return;
    }
    // 设置情人
    void setLover(std::string lover)
    {
        m_lover = lover;
    }

private:
    // 姓名 可读 可写
    std::string m_name;

    // 年龄 可读可写 如果想修改，必须在0-150
    int m_age;

    // 情人 只写
    std::string m_lover;
};

int main()
{
    Person p;
    p.setName("张三");
    std::cout << "姓名为： " << p.getName() << std::endl;
    p.setAge(1000);
    std::cout << "年龄为： " << p.getAge() << std::endl;
    p.setLover("苍井");
    return 0;
}
```

## 构造函数和析构函数
对象的初始化和清理

由编译器自动调用，完成对象的初始化和清理

我们若不提供构造和析构，编译器会提供，但编译器提供的构造和析构是空实现

* 构造函数语法`类名(){}`
* 析构函数语法`~类名(){}`
  
```c++
// 对象的初始化和清理

class Person
{
public:
    // 1、构造函数
    // 没有返回值，不用写void
    // 函数名与类名相同
    // 构造函数可以有参数，可以发生重载
    // 创建对象的时候，构造函数会自动调用，而且只会调用一次
    Person()
    {
        std::cout << "Person 构造函数的调用" << std::endl;
    }
    // 2、析构函数 进行清理的操作
    // 没有返回值，不写void
    // 函数名和类名相同，在名称前加~
    // 析构函数不可以有参数，不可以发生重载
    // 对象在销毁前 会自动调用析构函数，而且只会调用一次
    ~Person()
    {
        std::cout << "Person 析构函数调用" << std::endl;
    }
};

// 构造和析构都是必须有的实现，如果我们自己不提供，编译器会提供一个空实现的构造和析构
void test01()
{
    Person p; // 栈上的数据在函数结束后被释放会调用析构函数
    std::cout << "---------" << std::endl;
}

int main()
{
    test01();
    Person P;
    return 0;
}
```
>Person 构造函数的调用
>\---------
>Person 析构函数调用
>Person 构造函数的调用
>Person 析构函数调用
### 构造函数的分类及调用
1. 分类
   * 按照参数分类 无参构造、有参构造
   * 按照类型分类 普通构造、拷贝构造
2. 调用
   * 括号法
   * 显示法
   * 隐式转换法
```c++
// 1构造函数的分类及调用
// 分类
//  按照参数分类 无参构造 有参构造
//  按照类型分类 普通构造 拷贝构造
class Person
{
public:
    Person()
    {
        std::cout << "Person的无参构造(默认构造)函数调用" << std::endl;
    }
    Person(int a)
    {
        age = a;
        std::cout << "Person的有参构造函数调用" << std::endl;
    }
    // 拷贝构造
    Person(const Person &p)
    {
        // 将传入的人身上的所有属性，拷贝到我身上
        age = p.age;
        std::cout << "Person的拷贝构造函数调用" << std::endl;
    }
    ~Person()
    {
        std::cout << "Person的析构函数调用" << std::endl;
    }
    int age;
};

// 调用
void test01()
{
    // 1、括号法
    Person p; // 默认构造函数调用
    std::cout << "----------" << std::endl;
    Person p2(10);
    std::cout << "----------" << std::endl;
    Person p3(p2);
    std::cout << "----------" << std::endl;
    // 注意事项1
    // 调用默认构造函数时，不要加()
    // Person p();
    // 因为上面这行代码编译器会认为是函数的声明
    std::cout << p2.age << std::endl;
    std::cout << p3.age << std::endl;
    std::cout << "----------" << std::endl;

    // 2、显示法
    Person p11;
    Person p22 = Person(10);
    Person p33 = Person(p2);
    std::cout << "----------" << std::endl;
    Person(10); // 匿名对象 特点:当前行执行结束后,系统会立即回收匿名对象（除非给它赋予名字）
    std::cout << "----------" << std::endl;
    // 注意事项2
    // 不要利用拷贝构造函数初始化匿名对象
    // Person(p2);
    // 由于等价于 Person p2; 重定义

    // 3、隐式转换法
    Person p4 = 10; // 相当于写了Person p4=Person (10);
    Person p5 = p4;
    std::cout << "----------" << std::endl;
}

int main()
{
    test01();

    return 0;
}
```
### 拷贝构造函数的调用时机
* 使用一个已经创建完毕的对象来初始化一个新对象
* 值传递的方式给函数参数传值
* 以值方式返回局部对象
```c++
// 拷贝构造函数调用时机

class Person
{
public:
    Person()
    {
        std::cout << "Person默认构造函数调用" << std::endl;
    }
    ~Person()
    {
        std::cout << "Person析构函数调用" << std::endl;
    }
    Person(int age)
    {
        m_Age = age;
        std::cout << "Person有参构造函数调用" << std::endl;
    }
    Person(const Person &p)
    {
        m_Age = p.m_Age;
        std::cout << "Person拷贝构造函数调用" << std::endl;
    }
    int m_Age;
};
// 1、使用一个已经创建完毕的对象来初始化一个新对象
void test01()
{
    Person p1(20);
    Person p2(p1);
    std::cout << p2.m_Age << std::endl;
    std::cout << "--------" << std::endl;
}
// 2、值传递的方式给函数参数传值
void doWork(Person p)
{
}

void test02()
{
    std::cout << "--------" << std::endl;
    Person p;
    doWork(p); // 调用函数的值传递运用的是拷贝函数创建形参
}
// 3、值方式返回局部对象
Person doWork2()
{
    Person p1;
    std::cout << &p1 << std::endl;
    return p1;
}
void test03()
{
    std::cout << "--------" << std::endl;
    Person p = doWork2(); // 在gcc中，返回值不会运用拷贝构造函数
    std::cout << &p << std::endl;
}

int main()
{
    test01();
    test02();
    test03();

    return 0;
}
```
*值得注意的是，在很多编译器中，第三种拷贝时机已经不再使用*
### c++提供构造函数的规则
**创建类之后，c++会默认提供无参构造，有参构造，拷贝构造。**
* 编写无参构造之后，仍提供有参构造，拷贝构造
* 编写有参构造之后，不再提供无参构造，提供拷贝构造
* 编写拷贝构造之后，不再提供无参构造，拷贝构造

### 深拷贝和浅拷贝
**C++提供的拷贝构造采用浅拷贝**
类中在堆区开辟数据在析构时可能有问题
深拷贝可以解决
```c++
class Person
{
public:
    Person()
    {
        std::cout << "Person的默认构造函数调用" << std::endl;
    }
    Person(int age, int height)
    {
        std::cout << "Person的默认构造函数调用" << std::endl;
        m_Age = age;
        m_Height = new int(height);
    }
    Person(const Person &p)
    {
        m_Age = p.m_Age;
        //m_Height=p.m_Height 浅拷贝
        m_Height = new int(*p.m_Height); // 深拷贝，在堆区开辟空间，使拷贝的指针指向另一空间
    }
    ~Person()
    {
        // 析构函数 将堆区开辟数据做释放操作
        if (m_Height != NULL)
        {
            delete m_Height; // 浅拷贝带来问题 堆区内存重复释放  利用深拷贝解决
            m_Height = NULL;
        }
        std::cout << "Person的析构函数调用" << std::endl;
    }

    int m_Age;     // 年龄
    int *m_Height; // 身高
};
void test01()
{
    Person p1(18, 160);
    std::cout << "p1的年龄为 " << p1.m_Age << "身高为 " << *p1.m_Height << std::endl;
    Person p2(p1);
    std::cout << "p2的年龄为 " << p2.m_Age << "身高为 " << *p1.m_Height << std::endl;
}

int main()
{
    test01();

    return 0;
}
```

## 运算符重载
对于一些运算符`+-*/`等赋予新的意义

可以运用成员函数来进行自定义数据类型的运算

### 四则运算符重载
当这个成员函数名为`operator+`时，运算由原来的
```c++
a.成员函数名(b);
//简化为
a+b;
```
全局全局函数也可以通过`operator+`重载
```c++
class Person
{
public:
    int m_a = 10;
    int m_b = 10;
};
Person operator+(Person a, Person b)
{
    Person c;
    c.m_a = a.m_a + b.m_a;
    c.m_b = a.m_a + b.m_b;
    return c;
}
int main()
{
    Person a, b, c;
    c = a + b;
    std::cout << c.m_a << " " << c.m_b << std::endl;
    return 0;
}
```
这种方法适用于四则运算
### 左移运算符重载
* 首先要求需要是全局函数
* 传入参数左侧是`ostream`类的cout**引用**
* 返回也是`ostream`类的cout**引用**
```c++
using namespace std;
class Person
{
public:
    // 利用成员函数重载 左移运算符
    // 通常不会利用成员函数重载左移运算符 无法实现cout在左
    // void operator<<(std::cout)
    int m_a = 10;
    int m_b = 10;
};
ostream &operator<<(ostream &cout, Person p)
{
    cout << "m_A= " << p.m_a << " m_B= " << p.m_b;
    return cout;
}

int main()
{
    Person p;
    cout << p << endl;

    return 0;
}
```
### 自增运算符重载
* **前缀递增返回值要引用（始终对同一数据递增）**
* **后置递增不可返回引用**
```c++
using namespace std;
class MyInteger
{
    friend ostream &operator<<(ostream &cout, MyInteger a);

public:
    MyInteger()
    {
        m_Num = 0;
    }

    MyInteger & operator++()//前置递增
    {
        m_Num++;
        return *this;
    }
    MyInteger  operator++(int) //后置递增 int代表占位参数，用于区分前置，后置递增 
    {
        MyInteger temp(*this);
        m_Num++;
        return temp;
    }

private:
    int m_Num;
};
ostream &operator<<(ostream &cout, MyInteger a)
{
    cout << a.m_Num;

    return cout;
}

void test01()
{
    MyInteger myint;
    cout << ++myint << endl;
    cout << myint << endl;
}

int main()
{
    test01();
    return 0;
}
```
### 赋值运算符重载
类中c++提供了默认的operator=函数用于值拷贝（浅）

如要改为深拷贝
和拷贝构造的深拷贝类似

* 传参要求&
* 返回值要求& 
```c++
// 赋值运算符的重载
class Person
{
public:
    Person(int age)
    {
        m_Age = new int(age);
    }
    ~Person()
    {
        if (m_Age != NULL)
        {
            delete m_Age;
            m_Age = NULL;
        }
    }
    Person &operator=(Person &a)
    {
        // m_Age=a.m_Age; c++提供的
        if (m_Age != NULL)
        {
            delete m_Age;
            m_Age = NULL;
        }
        m_Age = new int(*a.m_Age);
        return *this;
    }

    int *m_Age;
};
void test01()
{
    Person p1(18);
    Person p2(18);
    Person p3(18);
    p3 = p2 = p1; // 赋值操作 运用c++提供的值拷贝
    std::cout << "p1的年龄为 " << *p1.m_Age << std::endl;
    std::cout << "p2的年龄为 " << *p2.m_Age << std::endl;
    std::cout << "p3的年龄为 " << *p3.m_Age << std::endl;
}

int main()
{
    test01();

    return 0;
}
```

## 友元

---

* 让**类外一些函数或类进行访问私有属性**
* 关键字为**friend**

三种实现

* 全局函数做友元
* 类做友元
* 成员函数做友元

### 全局函数做友元

---

* `friend`加函数声明

```c++
#include <iostream>
#include <cstdio>
#include <string>

using namespace std;

// 建筑物类
class Building
{
    // goodGay全局函数是 Building好朋友，可以访问Building中私有成员
    friend void goodGay(Building *building);

public:
    Building()
    {
        m_SittingRoom = "客厅";
        m_BedRoom = "卧室";
    }

public:
    string m_SittingRoom; // 客厅

private:
    string m_BedRoom; // 卧室
};

// 全局函数
void goodGay(Building *building)
{
    cout << "好基友全局函数 正在访问:" << building->m_SittingRoom << endl;
    cout << "好基友全局函数 正在访问:" << building->m_BedRoom << endl;
}

void test01()
{
    Building building;
    goodGay(&building);
}

int main()
{
    test01();
    return 0;
}
```

### 友元类

---

* `friend`加类声明

```c++
#include <iostream>
#include <cstdio>
#include <string>

using namespace std;

// 类做友元

class Building;

class GoodGay
{
public:
    GoodGay();
    void visit(); // 参观函数 访问Building中的属性

    Building *building;
};

class Building
{
    //GoodGay类是本类的好朋友，可以访问本类中私有成员
    friend class GoodGay;

public:
    Building();

public:
    string m_SittingRoom; // 客厅

private:
    string m_BedRoom; // 卧室
};

// 类外写成员函数
Building::Building()
{
    this->m_SittingRoom = "客厅";
    this->m_BedRoom = "卧室";
}

GoodGay::GoodGay()
{
    // 创建建筑物对象
    this->building = new Building;
}

void GoodGay::visit()
{
    cout << "好基友类 正在访问:" << this->building->m_SittingRoom;
    cout << "好基友类 正在访问:" << this->building->m_BedRoom;
}
void test01()
{
    GoodGay gg;
    gg.visit();
}

int main()
{
    test01();
    return 0;
}
```

### 成员函数做友元

* `friend`加成员函数声明

```c++
#include <iostream>
```

### 继承方式

---

* 公共继承 权限不变
* 保护继承 权限全变为保护
* 私有继承 权限全变为私有

*私有权限无法直接继承*

```c++
#include <iostream>
#include <cstdio>

using namespace std;

// 继承方式

// 公共继承
class Base1
{

public:
    int m_A;

protected:
    int m_B;

private:
    int m_C;
};

class Son1 : public Base1
{
public:
    void func()
    {
        this->m_A = 10; // 父类中公共权限成员到子类中仍为公共
        this->m_B = 20; // 父类中保护权限成员到子类中仍为保护
        // this->m_C = 30; // 父类中私有权限成员不可访问
    }
};

void test01()
{
    Son1 s1;
    s1.m_A = 100;
    // s1.m_B = 100; // 在Son1中，m_B是保护权限 类外访问不到
}

// 保护继承
class Base2
{

public:
    int m_A;

protected:
    int m_B;

private:
    int m_C;
};

class Son2 : protected Base2
{
public:
    void func()
    {
        this->m_A = 10; // 父类中公共权限成员到子类中为保护
        this->m_B = 20; // 父类中保护权限成员到子类中为保护
        // this->m_C = 30; // 父类中私有权限成员不可访问
    }
};

void test02()
{
    Son2 s1;
    // s1.m_A = 1000; // 在Son2中，m_A是保护权限 类外访问不到
    // s1.m_B = 1000; // 在Son2中，m_B是保护权限 类外访问不到
}

// 私有继承
class Base3
{

public:
    int m_A;

protected:
    int m_B;

private:
    int m_C;
};

class Son3 : private Base3
{
public:
    void func()
    {
        this->m_A = 10; // 父类中公共权限成员到子类中为私有
        this->m_B = 20; // 父类中保护权限成员到子类中为私有
        // this->m_C = 30; // 父类中私有权限成员不可访问
    }
};

void test03()
{
    Son3 s1;
    // s1.m_A = 1000; // 在Son3中，m_A是私有权限 类外访问不到
    //  s1.m_B = 1000; // 在Son3中，m_B是私有权限 类外访问不到
}

class GrandSon3 : public Son3
{
public:
    void func()
    {
        //this->m_A = 1000; //到了Son3中m_A变为私有，及时儿子也访问继承不到
        //this->m_B = 1000;
    }
};

{
    return 0;
```
